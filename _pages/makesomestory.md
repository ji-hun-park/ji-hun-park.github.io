## 1. 프로젝트 개요
대학교 전자상거래 강의의 과제 제출용으로 제작했습니다.

### 📋프로젝트 정보

* **장르 / 플랫폼:** 2D 메이크 게임 / PC (Single)
* **개발 인원:** 1인 개발, 클라이언트 및 시스템 담당
* **제작 기간:** 2024.10 ~ 2024.10 약 2주 소요(기존 소스 재활용)
* **사용 엔진 / 언어 / 환경:** Unity6 (6000.0.32f1) / C# / VSCode

## 2. 게임 소개
**Unity 2D 이야기 생성 어플리케이션**  
이미지와 컨셉을 입력해 이야기를 생성하는 앱입니다.
